Подробно о безопасности

Безопасность

  • Платежный сервис PAY2CARD обладает сертификатом соответствия международному стандарту безопасности PСI DSS (Payment Card Industry Data Security Standard). Данный стандарт разработан для обеспечения информационной безопасности платежных операций с участием банковских карт и является обязательным требованием для всех компаний, работающих с международными платежными системами Visa и MasterCard.
  • Использование сертификата SSL (Secure Sockets Layer) обеспечивает безопасность передачи данных. Обмен информацией через сайт происходит по защищенному каналу, что гарантирует конфиденциальность и сохранность сведений.
  • Для подтверждения платежа используется технология 3-D Secure. В процессе оплаты владельцу банковской карты отправляется SMS с одноразовым кодом. Его нужно ввести в диалоговое окно на сайте – для подтверждения операции. Таким образом, все платежи производятся только с согласия владельца карты.
  • При совершении операции клиент вводит трехзначный код CVC/CVV, подтверждающий подлинность карты и тот факт, что платеж совершает лицо, физически владеющее картой. Данный код НЕ сохраняется в Платежном сервисе А3 – каждый раз при осуществлении оплаты его нужно вводить повторно.
  • Также для подтверждения ряда действий на сайте Платежного сервиса А3, таких как смена пароля, редактирование личных данных и других, применяется идентификация пользователя с помощью одноразового пароля, который отправляется на номер мобильного телефона. Сведения об одноразовых паролях не хранятся в Платежном сервисе А3.
